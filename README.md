# Mejoras · Planilla Madre — Documentación de Flota

Página web con la revisión técnica y las **14 mejoras propuestas** para la planilla
`PLANILLA MADRE - DOCUMENTACION FLOTA.xlsx` (control de vencimientos de documentos de tractos y ramplas).

👉 La página es un único archivo `index.html`, sin dependencias externas.

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub (p. ej. `mejoras-planilla-flota`).
2. Sube el contenido de esta carpeta (`index.html` y `README.md`).
3. En el repo: **Settings → Pages → Source: `main` / carpeta `/root`** y guarda.
4. En 1–2 minutos la página queda publicada en
   `https://<tu-usuario>.github.io/mejoras-planilla-flota/`.

## Contenido

- **A. Integridad de datos** — estados inconsistentes, encabezados que rompen fórmulas, historial fragmentado.
- **B. Fórmulas y lógica** — plazos "quemados", doble cálculo de estado, búsquedas frágiles.
- **C. Rendimiento** — filas vacías, listas duplicadas, columnas poco descriptivas.
- **D. Usabilidad y escala** — formato condicional, panel resumen, validación de patente, estrategia a futuro.
- **E. Hoja de ruta** — de lo más rápido a lo estructural.
